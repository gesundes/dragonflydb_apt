⚠️ Unofficial Repository: This is an unofficial mirror of DragonflyDB packages. For official packages, visit DragonflyDB Releases.

Quick Setup

Add this repository to your system:

# Add GPG key
curl -fsSL https://gesundes.github.io/dragonflydb_apt/pubkey.gpg | sudo gpg --dearmor -o /usr/share/keyrings/dragonflydb.gpg

# Add repository
echo "deb [signed-by=/usr/share/keyrings/dragonflydb.gpg] https://gesundes.github.io/dragonflydb_apt stable main" | sudo tee /etc/apt/sources.list.d/dragonflydb.list

# Update package list
sudo apt update

# Install DragonflyDB
sudo apt install dragonfly
